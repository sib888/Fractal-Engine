# Fractal Engine
Многопоточный консольный движок на Java для генерации и визуализации фрактальных множеств (Мандельброт, Жюлиа, Горящий Корабль) с поддержкой глубокого масштабирования (Zoom).

## Ключевые фичи и технологии
* **Java Concurrency (`ExecutorService`, `Future`):** Вычисления полностью распараллелены. Картинка нарезается на независимые задачи по строкам и распределяется по ядрам процессора, что ускорило генерацию более чем в 6 раз по сравнению с однопоточным режимом.
* **Java NIO.2:** Реализовано умное сохранение в файлы. Движок автоматически создает структуру папок и защищает старые рендеры от перезаписи, динамически добавляя индексы к именам файлов.
* **Абстракция:** Логика вычисления фракталов абстрагирована через интерфейсы, что позволяет добавлять новые фракталы без изменения кода многопоточного планировщика.

## Результаты оптимизации (Бенчмарк)
Тестирование проводилось на изображении разрешения 1920x1080 при `maxIter = 3000`:
* **В один поток:** ~3100 мс
* **В многопоточном режиме (Executor Service):** ~465 мс
